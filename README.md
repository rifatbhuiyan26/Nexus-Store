# 👟 NEXUS - Premium Performance Gear Store

NEXUS is a stylish and modern sneaker e-commerce frontend project built with **HTML**, **Tailwind CSS**, **custom CSS**, **JavaScript**, **JSON**, and a small **Python sync script**.  
It provides a premium shopping experience with category browsing, product listing, product details, cart management, and checkout flow.

## ✨ Project Overview

This project is designed as a sleek online sneaker store where users can explore premium shoes, add products to cart, view cart totals, and place an order through a smooth checkout experience.

The UI follows a futuristic dark theme with glassmorphism effects, bold typography, and interactive product cards.

---

## 🚀 Features

- ✅ Premium landing page
- ✅ Category selection page
- ✅ Product listing page
- ✅ Product details page
- ✅ Add to cart functionality
- ✅ Cart page with total price calculation
- ✅ Remove item from cart
- ✅ Checkout form
- ✅ Order confirmation flow
- ✅ Local storage based cart system
- ✅ JSON-based product data
- ✅ Python script for syncing products into frontend
- ✅ Responsive modern UI
